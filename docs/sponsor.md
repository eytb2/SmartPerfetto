## AI Credits / Token 厂商赞助

SmartPerfetto 的开发、回归测试和公开演示会持续消耗模型 token。欢迎模型厂商、
API 平台、聚合平台或代理服务提供 API credits、token 套餐、试用账号或
开发者专项额度。

赞助额度会优先用于：

- Provider 兼容性验证
- tool calling / streaming 行为回归
- 公开 demo trace 和教程
- Issue 复现和开源维护
- 不同模型在真实 trace 分析任务中的可用性验证

可提供的合作形式：

- 在本页面展示赞助厂商名称和链接
- 在实际跑通后补充 provider 配置说明
- 发布 SmartPerfetto 真实 trace 分析案例
- 为 SmartPerfetto 用户提供试用码或开发者额度

赞助不会影响 SmartPerfetto 的默认 provider 选择、兼容性结论或技术建议。
Provider 行为会基于实际验证结果记录；如果某个模型不适合 trace 分析、工具调用
或长链路推理，也会如实说明。

如需提供 AI credits / token 赞助，请通过微信联系：`553000664`。
