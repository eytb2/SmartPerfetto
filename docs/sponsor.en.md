Assistant for Android Perfetto trace analysis. Its recurring cost is mostly
model tokens used for provider compatibility checks, public demos, trace
regression, and issue reproduction.

If SmartPerfetto helps your work, you can support the project through personal
donations, AI credits sponsorship, commercial support, or commercial licensing.

## AI Credits / Token Sponsorship

SmartPerfetto development, regression testing, and public demos continuously
consume model tokens. Model providers, API platforms, aggregators, and proxy
services are welcome to sponsor API credits, token packages, trial accounts, or
developer credits.

Sponsored credits will be used for:

- Provider compatibility validation
- Tool-calling and streaming regression checks
- Public demo traces and tutorials
- Issue reproduction and open-source maintenance
- Real trace-analysis usability checks across models

Possible collaboration formats:

- List the sponsor name and link on this page
- Add provider configuration notes after successful validation
- Publish real SmartPerfetto trace-analysis case studies
- Offer trial codes or developer credits to SmartPerfetto users

Sponsorship does not affect SmartPerfetto's default provider choice,
compatibility conclusions, or technical recommendations. Provider behavior is
documented based on actual validation results. If a model is not a good fit for
trace analysis, tool calling, or long-horizon reasoning, that will be stated
clearly.
